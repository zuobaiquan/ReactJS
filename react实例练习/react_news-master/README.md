# React_news
使用react开发实现pc端和移动端新闻头条网站。

博客地址：https://www.jianshu.com/p/25cabb061850

## 技术栈

前端UI框架:Ant Design

前端：使用React前端框架。react-router、react-responsive（响应式设计）

前后端数据交互：fetch
