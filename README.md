# React

好处：

1.组件化——分工、合作
2.虚拟DOM——性能高
3.跨平台——移动端



React全家桶(技术栈)
1.React主体
2.WebPack：grunt、gulp自动化构建工具
3.Flex：布局
4.React-Route：路由
5.Redux：View层
6.Mocha：测试
7.Istanbul：覆盖率





JSX：

2.babel

